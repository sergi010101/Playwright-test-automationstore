Проект содержит автотесты для сайта [automationteststore.com](https://automationteststore.com).

## Сквозное тестирование "от регистрации до покупки"
**Файл:** `tests/test-3.spec.ts`

### Что проверяет сценарий:
1. **Регистрация**: Создание нового аккаунта с заполнением всех обязательных полей.
2. **Уникальность данных**: Реализована генерация **рандомных значений** для полей `email` и `login`. Это гарантирует успешное прохождение теста при каждом запуске, исключая ошибку «пользователь уже существует».
3. **Покупка**: Добавление товара в корзину, переход к оформлению (Checkout) и подтверждение заказа.

## Запуск
```bash
npx playwright test tests/test-3.spec.ts --headed
```
